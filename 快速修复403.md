# 快速修复 403 Forbidden

## 🔍 问题

403 Forbidden = 认证失败

标准 ACR 需要使用**账号密码**，不是 AccessKey。

## ✅ 立即修复（2 步）

### 步骤 1: 获取 ACR 密码

1. 登录：https://cr.console.aliyun.com
2. 点击 **访问凭证**
3. 重置或查看密码
4. 复制密码

### 步骤 2: 更新 GitHub Secrets

访问：https://github.com/leizhuofan2005/dfs-mazesolver/settings/secrets/actions

**更新这两个 Secrets：**

1. **ALIYUN_USERNAME**
   - 删除旧的
   - 创建新的
   - 值：你的**阿里云账号邮箱**（例如：`hi312*****@aliyun.com`）

2. **ALIYUN_ACR_PASSWORD**
   - 删除旧的
   - 创建新的
   - 值：**ACR 服务密码**（从访问凭证中获取的）

### 步骤 3: 重新触发构建

在 GitHub Actions 页面点击 "Re-run jobs"

---

**重要**：必须使用账号邮箱和 ACR 密码，不是 AccessKey！
